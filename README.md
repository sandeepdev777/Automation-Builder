Step 0:

Note ‚ÄºÔ∏è the application uses Clerk for Authentication and User Management, therefore, you need to create Clerk account [here](https://dashboard.clerk.com/sign-up) and sets the CLERK_PUBLISHABLE_KEY and CLERK_SECRET_KEY environment variables in .env file.


Step 1:

Download or clone this repo by using the link below:

git clone https://github.com/sandeepdev777/Fuzzie.git

step 2:

Execute the following command in the root directory of the downloaded repo in order to install dependencies:

npm install

Step 3:

Execute the following command in order to run the development server locally:

npm run dev

Step 4:

Open http://localhost:3000 with your browser to see the result.
